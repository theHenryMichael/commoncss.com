Common CSS Declarations

COLOR

Color

Example Code:

For Red: p { color: red; }

  • Red
  • Orange
  • Yellow
  • Green
  • Blue
  • Purple/Violet
  • Pink

Background Color

Example Code:

For Red: p { background-color: red; }

  • Red
  • Orange
  • Yellow
  • Green
  • Blue
  • Purple/Violet
  • Pink

Font Styling

Font Families (Also Known as Typeface)

Example Code:

Serif: p { font-family: serif; }

Sans-Serif: p { font-family: sans-serif; }

Monospace: p { font-family: monospace; }

Cursive: p { font-family: cursive; }

Fantasy: p { font-family: fantasy; }

Callback Fonts: p { font-family: "Abril Fatface", sans-serif; }

  • This displays a "serif" typeface value
  • This displays a "sans serif" typeface value
  • This displays a "Monospace" typeface value
  • This displays a "Cursive" typeface value
  • This displays a "Fantasy" typeface value
  • This displays an "Abril Fatface" typeface value, but if the font is unavailable, it will call back to the next available "serif" font

Font Sizing

Example Code:

Size by Pixels: p { font-size: 25px; }

Size by Relativity: p { font-size: 1.6em; }

  • This sentence has a size of 25 pixels.
  • This sentence is 1.6 times its normal size.

Text Effects

Example Code:

Text Shadow Effect: p { text-shadow: 3px 3px grey; }

Text "Shade" Effect: p { text-shadow: 1px 1px 1px black; }

Box Shadow Effect: p { box-shadow: 10px 10px black; }

  • This text has a grey shadow effect.
  • This white text has a black shadow (shade) effect.
  • This box has a black shadow effect.

The Box Model

Margin

Example Code:

Margin: p { margin: 10px; }

There is also p { margin-left: 10px; } { margin-right: 10px; }

p { margin-top: 10px; } { margin-bottom: 10px; }

To make things simpler: p { margin: 10px 10px; }

The first 10px represents the top and bottom values. The second 10px represents the left and right values.

  • This box has a margin of 20 pixels all around it.

Padding

Example Code:

Padding: p { padding: 10px; }

There is also p { padding-left: 10px; } { padding-right: 10px; }

p { padding-top: 10px; } { padding-bottom: 10px; }

To make things simpler: p { padding: 10px 10px; }

The first 10px represents the top and bottom values. The second 10px represents the left and right values.

  • This box contains 10 pixels of padding all around the content (text).

Border

Example Code:

Regular Border: p { border: 3px; }

There is also p { border-left: 3px; } { border-right: 3px; }

  • The blue dots that surround this text are part of a Border command:
  • p { border: 5px dotted blue;
  • border-radius: 15px; }
